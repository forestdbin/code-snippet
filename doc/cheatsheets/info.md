# info cheatsheet

## key bindings

```
q       quits
H       lists all Info commands
h       starts the Info tutorial

<space> forward
<backspace> <delete> backward

n       next
p       previous
]
[

^l      redraw display
b       beginning
e       ending

u       up
t       top
d       dir
l       last
x       delete

m       menu
f       follow cross-reference
r       follow xref
i       index
g       goto
<tab>   next link

s       search
S       case sensitive search
R       re toggle
^s      I-search
^r      I-search backward
```

## tips

### change scroll behavior

```
~x describe-variable scroll-behavior
~x set-variable scroll-behavior Page Only
```
